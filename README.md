# bezdima2025

1. **Agent** - сервис с пошаговым описанием работы агента в AI Agents, [подробное руководство](https://github.com/ivanteyivan/bezdima2025/blob/main/agent/README.md).

2. **Backend** - FastAPI‑сервис для генерации и проверки тестов с использованием Foundation Model, [подробное руководство](https://github.com/ivanteyivan/bezdima2025/blob/main/backend/README.md).

3. **Frontend** - React-приложение с узлами работы, например, загрузка, оптимизация, генерация тестов, [подробное описание API](https://github.com/ivanteyivan/bezdima2025/blob/main/frontend/my-react-flow-app/README.md).

Особенности
- Быстрая стартовая структура
- Место для сценариев запуска, конфигураций и документации

Быстрый старт
1. Клонируйте репозиторий:
   git clone https://github.com/ivanteyivan/bezdima2025.git
2. Перейдите в папку проекта:
   cd bezdima2025
3. Следуйте инструкциям в соответствующих директориях (например, /src, /docs) для запуска или сборки.
